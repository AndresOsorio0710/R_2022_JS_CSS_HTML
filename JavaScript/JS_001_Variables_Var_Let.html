Variables

Var:

Utilizada para ambito global o de la funcion, se recomienda unsar en menor medida esta forma de declarar variables

Let:

Utilizada para ambito de bloque

Objeto window

El objeto window es el objeto que mapea toda la ventana del navegador

Constantes

Nos permite crear variables con valor Constantes

Objetos

Un objeto en una variable con diversos valores, se declara entre corchetes "{}" y este puede tener dentro otrtos objetos

Arreglos

Es un conjunto de valore, se declara entre llaves "[]"
